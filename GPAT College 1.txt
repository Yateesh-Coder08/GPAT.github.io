                <div class="prose prose-slate max-w-none text-slate-700 space-y-6 leading-relaxed">
                    <p class="text-lg">
                        The <strong>Graduate Pharmacy Aptitude Test (GPAT)</strong> is a highly competitive, national-level entrance examination conducted in India. It serves as the primary gateway for pharmacy graduates (B.Pharm) seeking admission into elite Master of Pharmacy (M.Pharm) programs across the country. 
                    </p>

                    <h3 class="text-2xl font-bold text-slate-900 mt-8 mb-3">Who Conducts GPAT?</h3>
                    <p>
                        Previously managed by the National Testing Agency (NTA), the responsibility of conducting GPAT has been shifted to the <strong>National Board of Examinations in Medical Sciences (NBEMS)</strong>. This board ensures a standardized, fair, and rigorous testing environment through a Computer-Based Test (CBT) format.
                    </p>

                    <h3 class="text-2xl font-bold text-slate-900 mt-8 mb-3">Why is GPAT So Important?</h3>
                    <ul class="list-disc pl-5 space-y-2">
                        <li><strong>Gateway to Premier Institutes:</strong> A good GPAT score is mandatory for admission into top-tier institutions like the National Institutes of Pharmaceutical Education and Research (NIPERs), Jamia Hamdard, ICT Mumbai, and BITS Pilani.</li>
                        <li><strong>AICTE PG Scholarship (The Stipend):</strong> Perhaps the most significant advantage of qualifying GPAT is the financial aid. GPAT-qualified candidates admitted to AICTE-approved institutions are eligible for a monthly stipend of <strong>₹12,400</strong> (totaling nearly ₹3 Lakhs over two years). This stipend effectively neutralizes the tuition fees for many colleges, and in some cases, creates a financial surplus for the student.</li>
                        <li><strong>Ph.D. Opportunities:</strong> A valid GPAT score is also highly respected when applying for doctoral (Ph.D.) programs and various Junior Research Fellowship (JRF) positions funded by the CSIR.</li>
                    </ul>

                    <h3 class="text-2xl font-bold text-slate-900 mt-8 mb-3">Exam Pattern & Syllabus</h3>
                    <p>
                        The GPAT exam evaluates a candidate's grasp of core pharmaceutical sciences. The syllabus broadly encompasses:
                    </p>
                    <div class="grid grid-cols-1 md:grid-cols-2 gap-4 mt-4">
                        <div class="bg-slate-50 p-4 rounded-xl border border-slate-100">
                            <h4 class="font-bold text-indigo-700">Pharmaceutics & Allied Subjects</h4>
                            <p class="text-sm mt-1">Formulation, drug delivery systems, physical pharmacy, and biopharmaceutics.</p>
                        </div>
                        <div class="bg-slate-50 p-4 rounded-xl border border-slate-100">
                            <h4 class="font-bold text-indigo-700">Pharmacology</h4>
                            <p class="text-sm mt-1">Mechanisms of drug action, therapeutic uses, and adverse effects.</p>
                        </div>
                        <div class="bg-slate-50 p-4 rounded-xl border border-slate-100">
                            <h4 class="font-bold text-indigo-700">Pharmaceutical Chemistry</h4>
                            <p class="text-sm mt-1">Organic chemistry, medicinal chemistry, and structural analysis.</p>
                        </div>
                        <div class="bg-slate-50 p-4 rounded-xl border border-slate-100">
                            <h4 class="font-bold text-indigo-700">Pharmacognosy</h4>
                            <p class="text-sm mt-1">Study of drugs derived from natural sources, plants, and traditional medicine.</p>
                        </div>
                    </div>

                    <h3 class="text-2xl font-bold text-slate-900 mt-8 mb-3">Career Prospects Post M.Pharm</h3>
                    <p>
                        Completing an M.Pharm from a top institution opens doors to advanced roles in Research and Development (R&D), Regulatory Affairs (ensuring drug compliance with FDA/CDSCO rules), Clinical Trials, Quality Assurance (QA/QC), and academia.
                    </p>

            const chatWidget = document.getElementById('aiChatWidget');
            const chatWindow = document.getElementById('aiChatWindow');
            const toggleChatBtn = document.getElementById('toggleChatBtn');
            const closeChatBtn = document.getElementById('closeChatBtn');
            const chatInput = document.getElementById('aiChatInput');
            const sendChatBtn = document.getElementById('sendChatBtn');
            const chatMessages = document.getElementById('aiChatMessages');
            const chatLoading = document.getElementById('aiChatLoading');

            let chatHistory = [];

            // Toggle visibility with smooth animation
            function toggleChat() {
                if (chatWindow.classList.contains('hidden')) {
                    chatWindow.classList.remove('hidden');
                    // Small delay to allow display:block to apply before animating opacity/scale
                    setTimeout(() => {
                        chatWindow.classList.remove('scale-95', 'opacity-0');
                        chatWindow.classList.add('scale-100', 'opacity-100');
                    }, 10);
                    chatInput.focus();
                } else {
                    chatWindow.classList.remove('scale-100', 'opacity-100');
                    chatWindow.classList.add('scale-95', 'opacity-0');
                    setTimeout(() => {
                        chatWindow.classList.add('hidden');
                    }, 300);
                }
            }

            toggleChatBtn.addEventListener('click', toggleChat);
            closeChatBtn.addEventListener('click', toggleChat);

            // Add a message bubble to the chat UI
            function appendMessage(text, isUser) {
                const msgDiv = document.createElement('div');
                msgDiv.className = `p-3 rounded-2xl shadow-sm text-sm max-w-[85%] leading-relaxed ${
                    isUser 
                    ? 'bg-indigo-600 text-white self-end rounded-tr-sm' 
                    : 'bg-white border border-slate-200 text-slate-700 self-start rounded-tl-sm'
                }`;
                
                // Extremely basic markdown formatting for bold and new lines
                let formattedText = text.replace(/\*\*(.*?)\*\*/g, '<strong>$1</strong>');
                formattedText = formattedText.replace(/\*/g, '•'); // Bullet points
                formattedText = formattedText.replace(/\n/g, '<br>');
                
                msgDiv.innerHTML = formattedText;
                chatMessages.appendChild(msgDiv);
                
                // Auto-scroll to bottom
                chatMessages.scrollTop = chatMessages.scrollHeight;
            }

            // Connect to Gemini API and get response
            async function sendMessage() {
                const text = chatInput.value.trim();
                if (!text) return;

                // 1. Add user message to UI
                appendMessage(text, true);
                chatInput.value = '';
                
                // 2. Add to conversation history
                chatHistory.push({ role: "user", parts: [{ text: text }] });

                // 3. Show loading indicator
                chatLoading.classList.remove('hidden');

                // 4. API Configuration (API Key is handled automatically by the environment)
                const apiKey = ""; 
                const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-3-flash-preview:generateContent?key=${apiKey}`;
                
                const systemPrompt = "You are an expert counselor for pharmacy students in India. You help them with GPAT exam details, college selection (like NIPERs, Jamia Hamdard, BITS), M.Pharm specializations (Pharmaceutics, Pharmacology, Regulatory Affairs, etc.), and career guidance. Keep answers concise, highly encouraging, helpful, and keep formatting simple (use markdown for bolding). Keep responses under 2-3 short paragraphs.";

                const payload = {
                    contents: chatHistory,
                    systemInstruction: { parts: [{ text: systemPrompt }] }
                };

                try {
                    const response = await fetch(apiUrl, {
                        method: 'POST',
                        headers: { 'Content-Type': 'application/json' },
                        body: JSON.stringify(payload)
                    });

                    const result = await response.json();
                    
                    if (result.candidates && result.candidates.length > 0) {
                        const aiText = result.candidates[0].content.parts[0].text;
                        
                        // Add AI response to history
                        chatHistory.push({ role: "model", parts: [{ text: aiText }] });
                        
                        // Add AI response to UI
                        appendMessage(aiText, false);
                    } else {
                        appendMessage("Sorry, I encountered an issue parsing the response. Please try again.", false);
                    }
                } catch (error) {
                    console.error("Chat API Error:", error);
                    appendMessage("Failed to connect to the counselor network. Please check your connection.", false);
                } finally {
                    // Hide loading indicator
                    chatLoading.classList.add('hidden');
                }
            }

            // Bind send events
            sendChatBtn.addEventListener('click', sendMessage);
            chatInput.addEventListener('keypress', (e) => {
                if (e.key === 'Enter') sendMessage();
            });
        });
